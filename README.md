# 打卡习惯 (DakaHabit) - 日常打卡软件

## 项目简介

打卡习惯是一款可爱风格的日常习惯养成应用，参考小日常、习惯点点等优秀应用设计。帮助用户建立和维护良好的生活习惯，通过科学的打卡机制和贴心的统计分析，让习惯养成变得简单有趣。

## 核心特性

### 🎯 习惯管理
- 创建自定义习惯或选择预设习惯模板
- 设置习惯的周期性（每日/每周/自定义）
- 配置每日打卡时间范围
- 习惯分类和标签管理
- 习惯难度和重要性设置

### ⏰ 智能打卡
- 灵活的打卡时间设置
- 打卡提醒和通知
- 快速打卡和批量打卡
- 打卡状态可视化
- 补卡和请假功能

### 📝 关联日志
- 习惯与日志多对多关联
- 丰富的日志记录（文字、图片、心情）
- 日志模板和快速记录
- 日志搜索和筛选

### 📊 数据统计
- 单个习惯详细统计
- 时间范围内的整体统计
- 习惯完成率和达成进度
- 连续打卡天数和最长记录
- 预计完成时间计算
- 可视化图表展示

### 🎨 可爱界面
- 参考习惯点点的可爱风格
- 温馨的色彩搭配
- 有趣的动画效果
- 个性化主题设置

## 技术栈

- **前端框架**: Flutter 3.x
- **状态管理**: Provider + ChangeNotifier
- **本地数据库**: SQLite (sqflite)
- **数据序列化**: json_annotation + json_serializable
- **路由管理**: go_router
- **UI组件**: Material 3 Design + 自定义组件
- **本地存储**: shared_preferences
- **通知**: flutter_local_notifications
- **图表**: fl_chart
- **图片处理**: image_picker
